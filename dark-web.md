--> Dark Web

"These are basic notes about the Dark Web."

--> What is the Dark Web?

The Dark Web is a part of the Internet that is not indexed by search engines and requires special software like the Tor Browser to access.

--> Features

- Provides anonymity.
- Cannot be accessed using normal web browsers.
- Uses .onion websites.

-->Uses

--> Legitimate Uses

- Privacy protection
- Anonymous communication
- Journalism

--> Illegal Uses

- Selling illegal goods
- Cybercrime activities
- Stolen data trading
