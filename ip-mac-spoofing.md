--> IP and MAC Spoofing

"These are basic notes on IP and MAC spoofing."

--> IP Spoofing

IP spoofing is a technique where an attacker changes the source IP address to hide their identity or impersonate another system.

-->Uses

- Hiding the attacker's identity
- Bypassing IP-based restrictions
- Launching network attacks

--> MAC Spoofing

MAC spoofing is the process of changing the MAC address of a network device.

--> Uses

- Testing network security
- Bypassing MAC filtering
- Protecting device privacy
