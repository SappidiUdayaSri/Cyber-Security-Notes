--> Sherlock

"Sherlock is an OSINT (Open Source Intelligence) tool used to find usernames across multiple social media platforms."

--> What Sherlock Does

- Searches for a username on different websites.
- Helps identify online accounts related to a target.
- Used during the reconnaissance (information gathering) phase.

--> Basic Command

python3 sherlock.py <username>

--> Example

python3 sherlock.py john123

--> Uses

- Username Enumeration
- Information Gathering
- Digital Footprinting
